A lightweight, performance-focused WordPress theme with modern design tokens and Core Web Vitals optimization.

== Description ==

Ascend is a minimalist WordPress theme designed with performance and accessibility in mind. It features:

- Lightweight CSS with design tokens for consistent styling
- Optimized JavaScript for minimal impact on page load
- Full support for WordPress block editor with custom styles
- Responsive design for all devices
- Semantic HTML markup
- Core Web Vitals best practices (< 300 KB total payload)
- Accessibility features (WCAG 2.1 AA)
- Multiple widget areas and menus
- Image optimization with lazy loading
- Editor styles for consistent content creation

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support custom menus? =
Yes, Ascend supports custom menus. You can set up menus in Appearance > Menus.

= Does this theme support custom logos? =
Yes, you can add a custom logo in Appearance > Customize > Site Identity.

= How do I add widgets? =
You can add widgets in Appearance > Widgets. The theme includes widget areas for the sidebar and footer.

= Is the theme mobile-friendly? =
Yes, Ascend is fully responsive and mobile-friendly.

= Does it support the block editor? =
Yes, Ascend fully supports the WordPress block editor with custom styling and design tokens.
