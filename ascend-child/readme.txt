=== Ascend Child ===

This is a child theme for the Ascend WordPress theme.

== Installation ==

1. Install and activate the parent theme "Ascend"
2. Install and activate this child theme
3. All parent theme functionality will be inherited
4. Add your custom modifications to style.css or functions.php

== Customization ==

- Add custom CSS to style.css
- Add custom PHP functions to functions.php
- Override parent theme templates by copying them to this directory

== Support ==

For support, please contact the theme author or visit the theme repository.
