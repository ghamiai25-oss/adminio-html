Installation

Method 1: WordPress Admin

  1. Download the theme ZIP file
  2. Go to Appearance ‚Üí Themes ‚Üí Add New
  3. Click Upload Theme
  4. Choose the ZIP file and click Install Now
  5. Click Activate

Method 2: FTP Upload

  1. Extract the theme ZIP file
  2. Upload the ascend folder to /wp-content/themes/
  3. Go to Appearance ‚Üí Themes
  4. Activate the Ascend theme
Requirements: WordPress 5.9+ and PHP 7.4+

Demo Import

Import demo content with one click to get started quickly.

Steps:

  1. Go to Appearance ‚Üí Demo Import
  2. Choose a demo (Corporate, Creative, or E-commerce)
  3. Click Import Demo
  4. Wait for the import to complete

What Gets Imported:

  • Sample pages and posts
  • Portfolio items
  • Widgets configuration
  • Theme customizer settings
  • Menus
Note: Demo import requires the WordPress Importer plugin

WooCommerce Integration

Ascend includes full WooCommerce support for creating online stores.

Setup WooCommerce:

  1. Install the WooCommerce plugin
  2. Complete the WooCommerce setup wizard
  3. Add products
  4. Configure shop pages

Features:

  • Product gallery with zoom and lightbox
  • Cart icon in header with live count
  • Responsive product grid
  • Optimized checkout process
  • Custom WooCommerce styles

Customization:

// Change products per page
add_filter( 'loop_shop_per_page', function() {
    return 16; // 16 products per page
} );

// Change number of columns
add_filter( 'loop_shop_columns', function() {
    return 4; // 4 columns
} );

Using Child Theme

A child theme is included for safe customization.

Installation:

  1. Upload both ascend and ascend-child folders
  2. Activate the Ascend Child theme
  3. Add customizations to child theme files

Customizing:

Add custom CSS to ascend-child/style.css:

/* Custom styles */
.site-header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

Add custom PHP to ascend-child/functions.php:

// Add custom function
function my_custom_function() {
    // Your code here
}
add_action( 'wp_footer', 'my_custom_function' );

Available Hooks

Ascend includes 20+ action hooks for developers:

Header Hooks:

  • ascend_before_header
  • ascend_header_start
  • ascend_header_end
  • ascend_after_header

Content Hooks:

  • ascend_before_main
  • ascend_main_start
  • ascend_main_end
  • ascend_after_main

Example Usage:

// Add content after header
add_action( 'ascend_after_header', function() {
    echo '<div class="custom-banner">Welcome!</div>';
} );
