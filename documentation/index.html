üì¶ Installation Guide

There are three ways to install the Ascend theme. Choose the method that works best for you:

Method 1: WordPress Admin Upload (Recommended)

This is the easiest method for most users.

1 Download Theme Files

Download the theme package from ThemeForest. You'll receive a file named ascend.zip or similar.

2 Access WordPress Admin

Log into your WordPress admin dashboard at yourdomain.com/wp-admin

3 Navigate to Themes

Go to Appearance ‚Üí Themes in your WordPress admin menu.

4 Click Add New

Click the "Add New" button at the top of the themes page.

5 Upload Theme

Click the "Upload Theme" button, then "Choose File", select your ascend.zip file, and click "Install Now".

6 Activate Theme

Once installed, click the "Activate" button to make Ascend your active theme.

‚úÖ Success!

Your theme is now installed and activated. Next, we recommend importing demo content to get started quickly.

Method 2: FTP Upload

For advanced users who prefer using FTP or have large theme files.

  1. Extract the ascend.zip file on your computer
  2. Connect to your server using an FTP client (like FileZilla)
  3. Navigate to /wp-content/themes/
  4. Upload the entire ascend folder
  5. Go to Appearance ‚Üí Themes in WordPress
  6. Find Ascend and click Activate

Method 3: cPanel File Manager

If you don't have FTP access but have cPanel access.

  1. Log into your hosting cPanel
  2. Open File Manager
  3. Navigate to public_html/wp-content/themes/
  4. Click Upload and select ascend.zip
  5. Right-click the uploaded file and select Extract
  6. Go to Appearance ‚Üí Themes in WordPress
  7. Activate the Ascend theme

üîß Troubleshooting

Common issues and their solutions:

Installation Issues

Problem: "The package could not be installed"

Solution:

  • Check that you're uploading the correct theme ZIP file
  • Ensure your server has enough disk space
  • Verify PHP memory limit is at least 64MB
  • Try FTP upload method instead

Problem: "Missing style.css stylesheet"

Solution:

  • You may have uploaded a double-zipped file
  • Extract the ZIP, then upload the inner ascend folder via FTP

Display Issues

Problem: Site looks broken after activation

Solution:

  1. Clear all caches (browser, WordPress, and plugin caches)
  2. Go to Settings ‚Üí Permalinks and click Save
  3. Disable all plugins temporarily to check for conflicts
  4. Check if your server meets minimum requirements

Problem: Header menu not showing

Solution:

  1. Go to Appearance ‚Üí Menus
  2. Create a menu or select existing menu
  3. Check "Primary Menu" under Display Location
  4. Click Save Menu

Demo Import Issues

Problem: Demo import times out

Solution:

  • Increase PHP max_execution_time to 300 seconds
  • Increase PHP memory_limit to 256MB
  • Contact your hosting provider to adjust these limits
  • Use the manual import method

Problem: Demo images not importing

Solution:

  • Ensure "Download and import file attachments" is checked
  • Check your uploads folder permissions (should be 755)
  • Verify your server can make external HTTP requests

WooCommerce Issues

Problem: Cart icon not showing in header

Solution:

  • Ensure WooCommerce plugin is activated
  • Clear all caches
  • Check if you're using the parent theme (should use child theme)

Problem: Product images too large/small

Solution:

  1. Go to WooCommerce ‚Üí Settings ‚Üí Products ‚Üí Display
  2. Adjust image dimensions
  3. Install "Regenerate Thumbnails" plugin
  4. Run thumbnail regeneration

Performance Issues

Problem: Site loading slowly

Solution:

  • Enable all performance settings in Customize ‚Üí Performance
  • Install a caching plugin (WP Rocket, W3 Total Cache)
  • Optimize images before uploading (use TinyPNG or similar)
  • Use a CDN for asset delivery
  • Check your hosting plan (upgrade if necessary)

Customization Issues

Problem: Customizer changes not saving

Solution:

  • Disable all caching plugins temporarily
  • Check browser console for JavaScript errors
  • Try a different browser
  • Disable other plugins to check for conflicts

Problem: Child theme styles not loading

Solution:

  • Ensure child theme's Template header says ascend
  • Verify child theme is activated (not parent theme)
  • Clear all caches
  • Check file permissions on child theme folder

Translation Issues

Problem: Text not translating

Solution:

  • Ensure translation plugin is installed (WPML, Polylang)
  • Check that the correct .mo file is in /wp-content/languages/themes/
  • Verify WordPress language is set correctly in Settings ‚Üí General

Getting More Help

If you can't resolve your issue:

  1. Check Site Health: Go to Tools ‚Üí Site Health for server information
  2. Enable Debug Mode: See the Developers section for instructions
  3. Contact Support: See the Support section below
üí° Pro Tip:

When contacting support, include your WordPress version, PHP version, active plugins list, and any error messages you're seeing.

‚ùì Frequently Asked Questions

General Questions

Q: Can I use Ascend on multiple sites?

A: The Regular License allows use on one website. For multiple sites, purchase additional licenses or an Extended License.

Q: Is Ascend compatible with page builders?

A: Yes, Ascend works with popular page builders including Elementor, Beaver Builder, and Gutenberg (WordPress Block Editor).

Q: Can I customize the theme colors?

A: Absolutely! You can customize all colors through the WordPress Customizer or by adding custom CSS to the child theme.

Q: Is the theme translation-ready?

A: Yes, Ascend is fully translation-ready with an included POT file. It's compatible with WPML, Polylang, and includes RTL support.

Technical Questions

Q: What PHP version do I need?

A: Minimum PHP 7.4, but we recommend PHP 8.0 or higher for best performance.

Q: Does Ascend work with WordPress 6.x?

A: Yes, Ascend is fully compatible with the latest WordPress version and is regularly updated.

Q: Can I use Ascend with WooCommerce?

A: Yes! Ascend includes full WooCommerce integration with custom styling and features.

Q: Is Ascend mobile-responsive?

A: Yes, Ascend is 100% responsive and mobile-first, looking great on all devices.

Customization Questions

Q: How do I add custom CSS?

A: Use the child theme's style.css file or add CSS in Appearance ‚Üí Customize ‚Üí Additional CSS.

Q: Can I modify theme templates?

A: Yes, copy templates to your child theme and modify them there. See the Developer Documentation section.

Q: How do I change fonts?

A: Go to Appearance ‚Üí Customize ‚Üí Ascend Theme Options ‚Üí Typography to select from included fonts.

Q: Can I create a mega menu?

A: While Ascend doesn't include a built-in mega menu, it's compatible with popular menu plugins like Max Mega Menu.

WooCommerce Questions

Q: How do I change the number of products per page?

A: Add a filter to your child theme's functions.php. See the WooCommerce section for code examples.

Q: Where does the cart icon appear?

A: The cart icon automatically appears in the header when WooCommerce is activated.

Q: Can I customize product page layout?

A: Yes, through WooCommerce settings, custom CSS, or by overriding WooCommerce templates in your child theme.

Support Questions

Q: How long do I get support?

A: Your purchase includes 6 months of support. This can be extended through ThemeForest.

Q: Do I get lifetime updates?

A: Yes! Theme updates are free forever, regardless of support status.

Q: What's included in support?

A: Bug fixes, installation help, configuration guidance, and theme feature questions. See the Support section for details.

Q: Can you customize the theme for me?

A: Custom development is not included in standard support, but we can provide recommendations for freelance developers if needed.
