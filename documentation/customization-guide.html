Ascend — Customization Guide

Theme author: ElevateWorks

Theme options (Customizer)

All core settings are available in the WordPress Customizer:

  • Go to Appearance → Customize
  • Open Ascend Theme Options

You can configure:

  • Header: layout, sticky behavior, colors
  • Footer: layout and copyright text
  • Typography: font choices and sizing
  • Social Media: profile links
  • Blog: sidebar and excerpt settings
  • Performance: lightweight enhancements

Homepage setup

  • Go to Settings → Reading
  • Choose Your homepage displays:
    • Your latest posts (blog-style homepage)
    • A static page (create/select a “Home” page)

Menus

  • Go to Appearance → Menus
  • Create a menu and assign it to:
    • Primary Menu
    • Secondary Menu (optional)
    • Footer Menu (optional)

Widgets

Ascend provides widget areas for common layouts:

  • Primary Sidebar
  • Footer Sidebar

Manage them via Appearance → Widgets.

Custom post types

After activating the theme you will see these content types in the admin menu:

  • Portfolio (with categories)
  • Team
  • Testimonials

WooCommerce

If you install and activate WooCommerce, Ascend automatically enables shop/product styling and templates.

WooCommerce settings are managed via WooCommerce → Settings.

Child theme (recommended for code edits)

If you plan to modify theme files, use a child theme to keep changes update-safe.

A child theme is included in the repository, but it is not part of the installable ascend.zip by default.

Support / troubleshooting

  • Clear any caching plugin after making style or layout changes.
  • Switch to a default theme temporarily to confirm plugin conflicts.
  • Keep WordPress and plugins updated.
