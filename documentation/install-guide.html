Ascend — Installation Guide

Theme author: ElevateWorks

Requirements

  • WordPress 5.9 or newer
  • PHP 7.4 or newer

Install the theme

  1. In your WordPress admin, go to Appearance → Themes.
  2. Click Add New → Upload Theme.
  3. Select ascend.zip and click Install Now.
  4. After installation completes, click Activate.

If you see “The package could not be installed”, re-download and ensure you are uploading ascend.zip (the installable theme ZIP), not the full ThemeForest package ZIP.

Recommended plugins

  • WooCommerce (optional) — enables shop/product templates and styling.

Import demo content

  1. Go to Tools → Import.
  2. Under WordPress, click Install Now (if prompted), then Run Importer.
  3. Upload demo-content.xml from the ThemeForest package.
  4. Assign the imported content to your desired user.
  5. Check Download and import file attachments (recommended), then run the import.

After import: set your homepage via Settings → Reading and configure menus via Appearance → Menus.

Basic setup checklist

  • Set Site Title/Tagline: Settings → General
  • Set homepage/blog page: Settings → Reading
  • Assign menus: Appearance → Menus
  • Configure widgets: Appearance → Widgets
  • Customize theme options: Appearance → Customize
