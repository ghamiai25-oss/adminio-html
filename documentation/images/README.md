## Image Specifications

### Format
- PNG format (preferred for UI screenshots)
- JPG format acceptable for photos

### Dimensions
- Minimum width: 1200px
- Maintain aspect ratio
- High resolution (2x for retina displays when possible)

### Quality
- Clear, crisp screenshots
- No personal information visible
- Clean browser interface (hide bookmarks bar if possible)
- Use light theme for WordPress admin
- Show relevant UI elements clearly

## How to Take Screenshots

### For WordPress Admin
1. Log into a WordPress site with Ascend installed
2. Navigate to the relevant page
3. Use a screenshot tool (Windows: Snipping Tool/Snip & Sketch, Mac: Cmd+Shift+4)
4. Capture the relevant area
5. Save with the corresponding filename

### For Frontend
1. Visit the frontend of your site
2. Ensure design is properly displayed
3. Clear browser toolbars if possible
4. Capture full width or specific sections as needed
5. Save with appropriate filename

### Tools Recommended
- **Windows:** Snipping Tool, Snip & Sketch, ShareX
- **Mac:** Built-in screenshot (Cmd+Shift+4), CleanShot X
- **Cross-platform:** Lightshot, Greenshot
- **Browser Extensions:** Awesome Screenshot, Full Page Screen Capture
